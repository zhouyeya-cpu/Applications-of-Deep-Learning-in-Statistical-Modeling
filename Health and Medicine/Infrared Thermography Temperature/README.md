## Dataset: Infrared Thermography Temperature

Source: UCI Machine Learning Repository <br>
Link: https://archive.ics.uci.edu/dataset/925/infrared+thermography+temperature+dataset <br>
UCI ID: 925 <br>
Domain: Health and Medicine <br>
Task: Tabular Regression <br>
Target Variable: `aveOralM`

The Infrared Thermography Temperature dataset contains temperature readings extracted from various locations in infrared thermal images, along with oral temperature measurements for each individual. The features include gender, age, ethnicity, ambient temperature, humidity, distance, and additional temperature readings from the thermal images. The prediction target used in this benchmark is `aveOralM`, which represents the oral temperature measured in monitor mode. In this benchmark, we use the dataset as a regression task to compare traditional statistical models, kernel methods, Gaussian process regression, tree-based models, and deep learning models.

## Dataset: Parkinsons Telemonitoring

Source: UCI Machine Learning Repository <br>
Link: https://archive.ics.uci.edu/dataset/189/parkinsons+telemonitoring <br>
UCI ID: 189 <br>
Domain: Health and Medicine / Remote Patient Monitoring <br>
Task: Tabular Regression <br>
Target Variable: `total_UPDRS`

The Parkinsons Telemonitoring dataset contains biomedical voice measurements collected from patients with early-stage Parkinson's disease, including demographic information, test time, and voice signal features such as jitter, shimmer, noise-to-harmonics ratio, harmonics-to-noise ratio, RPDE, DFA, and PPE. The prediction target used in this benchmark is `total_UPDRS`, which represents the clinician-rated total Unified Parkinson's Disease Rating Scale score. In this benchmark, we use the dataset as a regression task to compare traditional statistical models, kernel methods, Gaussian process regression, tree-based models, and deep learning models.

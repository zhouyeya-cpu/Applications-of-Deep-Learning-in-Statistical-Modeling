Dataset Information
Dataset Name: Seoul Bike Sharing Demand
Source: UCI Machine Learning Repository
UCI ID: 560
Domain: Business / Mobility Service
Task: Tabular Regression
Target Variable: Rented Bike Count
Dataset Link: https://archive.ics.uci.edu/dataset/560/seoul+bike+sharing+demand
Dataset Description

The Seoul Bike Sharing Demand dataset is a business-domain tabular regression dataset from the UCI Machine Learning Repository. The dataset contains hourly public bicycle rental demand from the Seoul Bike Sharing System, together with weather, date, season, holiday, and functional-day information.

The goal of this dataset is to predict the number of rented bikes required at each hour. This prediction task is important for bike-sharing system management because accurate demand prediction can help maintain a stable supply of rental bikes and reduce user waiting time.

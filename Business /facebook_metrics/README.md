Facebook Metrics Dataset
Dataset Information
Dataset Name: Facebook Metrics
Source: UCI Machine Learning Repository
UCI ID: 368
Domain: Business / Social Media Marketing
Task: Tabular Regression
Target Variable: Total Interactions
Dataset Link: https://archive.ics.uci.edu/dataset/368/facebook+metrics-
Dataset Description

The Facebook Metrics dataset is a business-domain tabular regression dataset from the UCI Machine Learning Repository. The data contain Facebook post performance metrics from a well-known cosmetics brand’s Facebook page. The posts were published during 2014.

The dataset includes 500 instances and 18 input features. It is associated with a regression task. In this benchmark, we use the dataset to predict post engagement performance, with Total Interactions as the target variable.

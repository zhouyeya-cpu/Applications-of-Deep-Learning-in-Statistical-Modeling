## Dataset: Beijing PM2.5

Source: UCI Machine Learning Repository  
Link: https://archive.ics.uci.edu/dataset/381/beijing+pm2+5+data  
UCI ID: 381  
Domain: Climate and Environment / Air Quality  
Task: Tabular Regression  
Target Variable: pm2.5

The Beijing PM2.5 dataset contains hourly air-quality and meteorological observations from Beijing. The features include calendar variables, dew point, temperature, pressure, wind direction, cumulative wind speed, snow, and rain. The prediction target used in this benchmark is `pm2.5`, representing PM2.5 concentration. The dataset has 43,824 rows, which is larger than the preferred 2k-row scale but still practical for model comparison, especially with row caps for cubic-time methods such as Gaussian process regression.

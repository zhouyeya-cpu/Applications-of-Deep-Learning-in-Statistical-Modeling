## Dataset: Airfoil Self-Noise

Source: UCI Machine Learning Repository  
Link: https://archive.ics.uci.edu/dataset/291/airfoil+self+noise  
UCI ID: 291  
Domain: Physics / Aerodynamics / Environmental Noise  
Task: Tabular Regression  
Target Variable: scaled-sound-pressure

The Airfoil Self-Noise dataset contains aerodynamic and acoustic measurements from NASA wind-tunnel experiments on airfoil blade sections. The input variables include frequency, angle of attack, chord length, free-stream velocity, and suction-side displacement thickness. The prediction target is `scaled-sound-pressure`, which represents the scaled sound pressure level in decibels. In this benchmark, we use the dataset as a regression task to compare traditional statistical models, kernel methods, Gaussian process regression, tree-based models, and deep learning models.

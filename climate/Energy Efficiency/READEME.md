Dataset: Energy Efficiency
Source: UCI Machine Learning Repository
Link: https://archive.ics.uci.edu/dataset/242/energy+efficiency
UCI ID: 242
Domain: Energy / Built Environment / Building Performance
Task: Tabular Regression
Target Variable: Y1 / Heating Load

The Energy Efficiency dataset contains simulated building-design variables, including relative compactness, surface area, wall area, roof area, overall height, 
orientation, glazing area, and glazing area distribution. The prediction target used in this benchmark is Y1, which represents heating load. In this benchmark, 
we use the dataset as a regression task to compare traditional statistical models, kernel methods, Gaussian process regression, tree-based models, and deep learning models.

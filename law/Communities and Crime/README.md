## Dataset: Communities and Crime

Source: UCI Machine Learning Repository  
Link: https://archive.ics.uci.edu/dataset/183/communities+and+crime  
UCI ID: 183  
Domain: Law / Crime / Public Safety  
Task: Tabular Regression  
Target Variable: ViolentCrimesPerPop  

The Communities and Crime dataset contains community-level socio-economic, law enforcement, and crime-related variables from the United States. The prediction target is `ViolentCrimesPerPop`, which represents per-capita violent crimes. In this benchmark, we use the dataset as a regression task to compare traditional statistical models, kernel methods, Bayesian regression, tree-based models, and deep learning models.

## Dataset: Communities and Crime Unnormalized

**Source:** UCI Machine Learning Repository  
**Link:** https://archive.ics.uci.edu/dataset/211/communities+and+crime+unnormalized  
**UCI ID:** 211  
**Domain:** Law / Crime / Public Safety  
**Task:** Tabular Regression  
**Target Variable:** `violentPerPop`

The Communities and Crime Unnormalized dataset contains community-level socio-economic, law enforcement, and crime-related variables from the United States. The dataset includes demographic, housing, police, and crime-related attributes.

In this benchmark, we use the dataset as a regression task to predict `violentPerPop`, which represents violent crimes per population. This dataset is included in the Law / Crime / Public Safety domain because it directly models community-level crime outcomes using social and law-enforcement-related variables.

Compared with the normalized Communities and Crime dataset, this unnormalized version contains raw or less-normalized community and crime-related variables, which provides another setting for evaluating model robustness on public-safety data.

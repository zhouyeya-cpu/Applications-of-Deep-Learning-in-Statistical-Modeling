Dataset Description

The Wine Quality dataset is a tabular dataset from the UCI Machine Learning Repository. It contains physicochemical test measurements of Portuguese “Vinho Verde” wine samples. The goal is to model wine quality based on measurable chemical properties.

Although the original dataset can be used for both classification and regression, this benchmark treats it as a regression task by predicting the wine quality score.

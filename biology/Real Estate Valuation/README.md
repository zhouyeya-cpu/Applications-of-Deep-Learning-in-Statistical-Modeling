Dataset Information
Dataset Name: Real Estate Valuation
Source: UCI Machine Learning Repository
UCI ID: 477
Domain: Business / Real Estate
Task: Tabular Regression
Target Variable: house price of unit area
Dataset Link: https://archive.ics.uci.edu/dataset/477/real+estate+valuation+data+set
Dataset Description

The Real Estate Valuation dataset is a tabular regression dataset from the UCI Machine Learning Repository. The data were collected from the real estate market in Sindian District, New Taipei City, Taiwan. The goal of this dataset is to predict the house price of unit area based on several real estate-related features.

This dataset is suitable for our benchmark because it is a clean tabular regression problem with numerical input features and a continuous target variable. It allows us to compare traditional statistical models, tree-based models, and deep learning models under the same regression pipeline.

Dataset Description

The Parkinsons Telemonitoring dataset is a biomedical tabular regression dataset from the UCI Machine Learning Repository. The dataset contains biomedical voice measurements from people with early-stage Parkinson’s disease. These measurements were collected as part of a telemonitoring study.

The goal of this dataset is to predict clinical Parkinson’s disease symptom scores from voice-related biomedical features. In this benchmark, we use motor_UPDRS as the regression target. The motor_UPDRS score measures the motor impairment severity of a patient.

The dataset contains 5,875 instances and multiple biomedical voice features. It is suitable for evaluating regression models on a real-world biomedical prediction task.

Dataset Description

The Abalone dataset is a biology-domain tabular dataset from the UCI Machine Learning Repository. The goal of this dataset is to predict the age of abalone from physical measurements. The age of abalone is usually determined by cutting the shell, staining it, and counting the number of rings under a microscope, which is time-consuming. Therefore, using easier-to-obtain physical measurements to estimate age provides a useful regression task.

The dataset contains 4,177 instances and 8 input features. It is associated with both classification and regression tasks. In this benchmark, we treat it as a regression problem by predicting the target variable Rings.
